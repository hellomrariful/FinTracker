# FinTracker 💰

A modern and intuitive personal finance tracking application built with Next.js 13, TypeScript, and Tailwind CSS. FinTracker helps you manage your income, expenses, and budget all in one place with a beautiful and responsive interface.

## 🚀 Features

- **Dashboard Overview**: Get a quick snapshot of your financial health
- **Income Management**: Track and categorize all your income sources
- **Expense Tracking**: Monitor and categorize your expenses
- **Budget Planning**: Set and track budgets for different categories
- **Recent Transactions**: View your latest financial activities
- **Dark Mode Support**: Easy on the eyes with dark mode support
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## 🛠️ Tech Stack

- **Framework**: Next.js 13 (App Router)
- **Language**: TypeScript/JavaScript
- **Styling**: Tailwind CSS
- **UI Components**: Shadcn UI
- **Icons**: Lucide Icons
- **State Management**: React Hooks

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/fintracker.git
   ```

2. Install dependencies:
   ```bash
   cd fintracker
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
